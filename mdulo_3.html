Módulo 3

Elaboración de documentos profesionales

Objetivo: Comprender el orden lógico de elaboración de un documento formal.

Aquí conocerás el orden lógico y las fases para crear un documento formal y profesional. Aprenderás a organizar el contenido, aplicar estilos, insertar elementos gráficos y referencias, y revisar la ortografía antes de finalizar el documento. Este módulo busca que comprendas cómo planificar y ejecutar correctamente cada etapa de un documento.

Descripción 

Aplicar estilos
Los estilos permiten dar uniformidad y profesionalismo a un documento (títulos, subtítulos, texto normal, citas, etc.).

Cómo hacerlo:

Selecciona el texto que quieras formatear.
Ve a la pestaña “Estilos” o “Formato” (según el programa: Word, Google Docs, LibreOffice).
Elige un estilo predefinido: Título 1, Título 2, Normal, Cita, Énfasis, etc.
Para personalizar estilos:

Haz clic derecho sobre el estilo → Modificar.
Cambia fuente, tamaño, color, interlineado y guardas los cambios.
 Ventaja: Si cambias el estilo, todo el texto que lo usa se actualizará automáticamente.


 Insertar elementos gráficos
Estos ayudan a ilustrar ideas y hacer el documento más atractivo.

Cómo hacerlo:

Ve a la pestaña “Insertar”.
Opciones comunes:

Imágenes: desde tu computadora o internet.
Formas: flechas, círculos, cuadros para resaltar información.
Gráficos: barras, líneas, pasteles, útiles para datos.
SmartArt o diagramas: esquemas o mapas conceptuales.
Ajusta tamaño, posición y estilo para integrarlo con el texto.
Tip: Usa subtítulos para cada gráfico para que quede claro de qué trata.

 Insertar referencias
Esto da soporte académico y evita el plagio.

Cómo hacerlo:

Para citas dentro del texto:

Ve a Referencias → Insertar cita (Word) o usa complementos como Zotero, Mendeley.
Escoge el formato: APA, MLA, Chicago, etc.
Para bibliografía o lista de referencias:

Después de citar, inserta la Bibliografía o Lista de referencias.
Se generará automáticamente según el estilo elegido.
Tip: Mantén todas las fuentes organizadas para actualizar fácilmente las referencias.


Revisar ortografía y gramática
Esto asegura que tu documento sea profesional y sin errores.

Cómo hacerlo:

Ve a Revisar → Ortografía y gramática (Word) o Herramientas → Ortografía y gramática (Google Docs).
El programa marcará errores: ortográficos (rojo) y gramaticales (verde o azul).
Corrige los errores uno por uno, revisando que el cambio tenga sentido.

Al finalizar 

lee tu documento en voz alta para detectar errores que el corrector no ve.
