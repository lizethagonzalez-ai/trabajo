Módulo 2

Uso y herramientas avanzadas

Objetivo: Aplicar correctamente los pasos para utilizar herramientas avanzadas del procesador de palabras

Este módulo está diseñado para que pongas en práctica el uso de las herramientas avanzadas del procesador de palabras. Aprenderás a insertar tablas, ecuaciones, citas y bibliografía, así como a personalizar cartas o documentos de manera profesional. La práctica constante permitirá consolidar tus habilidades técnicas y aumentar tu autonomía.

Descripción 

1. Insertar tablas
Las tablas son útiles para organizar información de forma clara.

Cómo hacerlo:

Ubica el cursor en el lugar donde quieres la tabla.
En la barra de herramientas, selecciona Insertar → Tabla.
Elige el número de filas y columnas que necesitas.
Una vez insertada, puedes:

Ajustar el ancho de las columnas y alto de las filas.
Combinar celdas si quieres títulos o secciones especiales.
Dar formato con colores, bordes o estilos de tabla para que sea más profesional.

2. Insertar ecuaciones
Sirve para mostrar fórmulas matemáticas, químicas o científicas.

Cómo hacerlo:

Coloca el cursor donde quieras la ecuación.
Ve a Insertar → Ecuación o en Word, usa el atajo Alt + =.
Aparecerá un cuadro donde puedes escribir:

Símbolos matemáticos (√, ∑, ∫, etc.)
Fracciones, exponentes, subíndices
Funciones trigonométricas o logaritmos
También puedes usar el editor de ecuaciones para dar formato profesional (alinear, numerar ecuaciones, etc.).

3. Insertar citas
Las citas permiten dar crédito a autores o fuentes.

Cómo hacerlo:

Coloca el cursor donde quieras la cita.
Ve a Referencias → Insertar cita → Agregar nueva fuente.
Completa los datos del autor, título, año, editorial y tipo de fuente (libro, artículo, web, etc.).
Una vez guardada, selecciona la fuente para insertarla.
Puedes elegir el estilo de cita (APA, MLA, Chicago, etc.) desde Estilo en la misma sección.

4. Insertar bibliografía
La bibliografía recopila todas las fuentes usadas.

Cómo hacerlo:

Una vez insertadas las citas en tu documento, ve a Referencias → Bibliografía.
Selecciona el estilo de bibliografía que prefieras (APA, MLA, Chicago…).
El programa genera automáticamente la lista de referencias con todas las fuentes citadas.
